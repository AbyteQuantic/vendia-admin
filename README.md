# VendIA Admin

Panel de administración web para VendIA.

## Tech Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Vitest** (testing)

## Setup Local

```bash
npm install
npm run dev    # http://localhost:3099
```

## Variables de Entorno

| Variable | Descripción |
|---|---|
| `BACKEND_URL` | URL del backend API (default: `http://localhost:8080`) |

## Build

```bash
npm run build
npm start
```
